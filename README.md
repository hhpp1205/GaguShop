# J FunitureStore
소규모 가구점을 위한 가구 홍보, 판매, 관리등을 한번에 할 수 있는 웹사이트 입니다.  


### Skill
Front : HTML, CSS, JavaScript, JQuery  
Back  : Java, Spring, Mybatis, MariaDB

### 기능
+ 가구 CRUD
+ 장바구니 추가, 삭제
+ 찜 추가, 삭제
+ 리뷰 <br>
